## ğŸ”§ Image Optimization (Recommended)

Your case study images are PNG format. Convert to AVIF/WebP for better performance:

```bash
# Install Squoosh CLI (if not already)
npm install -g @squoosh/cli

# Convert images
cd public/assets

# Convert to AVIF (best compression)
squoosh-cli --avif auto logo.jpg
squoosh-cli --avif auto email_ai.png
squoosh-cli --avif auto demo_chat1.png

# Convert to WebP (fallback)
squoosh-cli --webp auto logo.jpg
squoosh-cli --webp auto email_ai.png
squoosh-cli --webp auto demo_chat1.png
```

Then update components to use responsive srcset:
```tsx
<img
  src="/assets/email_ai.avif"
  srcset="/assets/email_ai.avif 1x, /assets/email_ai-2x.avif 2x"
  alt="AI Email Automation Agent"
  loading="lazy"
/>
```
